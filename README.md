# YouTube Downloader GUI

Application graphique pour télécharger des vidéos et audio depuis YouTube.

![Python](https://img.shields.io/badge/Python-3776AB?style=flat&logo=python&logoColor=white)

## 🎯 Fonctionnalités

- Téléchargement de vidéos YouTube en MP4
- Extraction audio en MP3
- Interface graphique intuitive avec Tkinter
- Choix du dossier de destination

## 📦 Installation

```bash
# Cloner le repository
git clone https://github.com/anaskiouaz/yt-dl-gui.git

# Installer les dépendances
cd yt-dl-gui
pip install -r requirements.txt

# Lancer l'application
python dlyoutube.py
```

## ⚠️ Dépendances

```
pytube
tkinter (inclus avec Python)
```

## 🚀 Utilisation

1. Lancer l'application
2. Coller l'URL de la vidéo YouTube
3. Choisir le format (MP4 ou MP3)
4. Sélectionner le dossier de destination
5. Cliquer sur "Télécharger"
